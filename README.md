# Scouting & Segmentação de Atletas com K-Means 

Este projeto utiliza Aprendizado Não Supervisionado para identificar perfis distintos de performance em um elenco, simulando um cenário de scouting profissional.

## O Problema
No futebol moderno, entender a similaridade entre jogadores é vital para substituições estratégicas e contratações assertivas (Replacement Analysis).

## Tecnologias e Técnicas
- **Algoritmo:** K-Means Clustering.
- **Otimização:** Método do Cotovelo (Elbow Method) para definição do K ideal.
- **Processamento:** StandardScaler para normalização de atributos físicos/químicos.
- **Visualização:** Seaborn e Matplotlib para mapeamento de perfis.

## Resultados
O modelo identificou 3 grupos principais:
1. **Perfil de Elite:** Atletas com alta combinação de intensidade e potência.
2. **Perfil Tático:** Jogadores de equilíbrio e suporte.
3. **Perfil Técnico:** Atletas com dominância em atributos específicos.
